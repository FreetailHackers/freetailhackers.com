# Freetail Hackers Website

This is a work in progress.

https://freetailhackers.com/

## Local Development

1. [Install Node.js](https://nodejs.org/en/download/) for your platform
2. `npm install -g node-sass` - installs globally <br>
   `npm install node-sass --save-dev` - installs locally
3. Start the Sass watcher: `node-sass --watch style.scss style.css` <br>
   If installed locally `npx node-sass --w style.scss style.css`
4. Open the `index.html` file

You can now make modifications to the `style.scss` file and it will be automatically recompiled. Refresh your browser window to see changes.

## Running the Website

If on vscode, install the Live Server extension and click Go Live on [index.html](./index.html)
