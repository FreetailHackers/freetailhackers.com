30 Day Git Challenge

Thanks for participating in the Freetail Hackers 30-Day Git Challenge! We understand it was tough to keep it going for 30 days straight, but we're really happy that you made it through. We accept commits that were pushed to any type of Git server, as long as you can provide us with an ability to publicly view your entire commit history in one place.

If you used GitHub, you can take advantage of our automated GitHub Streak Tracker so you can be certain there are no surprises. If you didn't use GitHub, please let us know in the form below, and we'll reach out to you if we have any questions.
